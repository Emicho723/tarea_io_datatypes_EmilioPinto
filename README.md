## Descripción del programa
Este programa en C++ solicita al usuario que ingrese su nombre y su edad. 
Luego muestra un mensaje personalizado en pantalla indicando el nombre 
y la edad ingresada.

El archivo principal del programa se llama deber1.cpp.

El objetivo del programa es demostrar el uso básico de:
- Entrada de datos con cin y getline
- Salida de datos con cout
- Uso de variables tipo string e int

## Instrucciones para compilar y ejecutar

### 1. Requisitos
- Tener instalado un compilador de C++ como g++

### 2. Compilar el programa
Abrir la terminal en la carpeta donde se encuentra el archivo deber1.cpp y ejecutar:

g++ deber1.cpp -o deber1

### 3. Ejecutar el programa

En Windows:
deber1.exe

En Linux o Mac:
./deber1

Después de ejecutarlo, el programa pedirá ingresar el nombre y la edad.
